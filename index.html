* Реализованы две вкладки, Add/Edit - для добавления и редактирования пользователей и List of User - для отображения списка всех пользователей. При нажатии на кнопку Add данные проходят валидации, в случае ошибки, появляется текстовое поле, с описанием ошибки. При исправлении ошибки и дальнейшей попытке добавления, текстовое поле пропадет, в случае успешной повторно валидации.
На вкладке List of user, пользователей можно сортировать по столбцам, для это нужно нажать на заголовок столбца.
Имеется возможность удалять пользователей, для этого на вкладке Add/Edit, нужно нажать Delete, около удаляемого пользователя. (Функционал редактирования пользователей на данный момент отсутствует, в силу того, что время отведенное на выполнение задания подходило к концу.)
